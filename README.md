# hyg

A small toy text editor written in plain POSIX C.

Called `hygge_editor` (or `hyg`) because writing it is a "hyggeprojekt" for me,
and it should really never be used for anything but "hyggeprojekter".

There remains a number of TODOs -- such as line soft-wrapping, escape code
handling, and a vim-like normal/command mode -- aswell as a *long* (but
implicit) list of non-goals, including, but definitely not limited to, support
for wide characters and syntax highlighting.
